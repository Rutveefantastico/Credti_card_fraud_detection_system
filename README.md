# 🛡️ Credit Card Fraud Detection System

A Machine Learning-based web application that detects fraudulent credit card transactions in real-time using Logistic Regression.

---

## 🚀 Live Demo
👉 [Click Here to Use the App](https://credticardfrauddetectionsystem-2m57zgcghnjwp8g2kkcxtz.streamlit.app)

---

## 📖 Project Overview

Credit card fraud is a major issue in financial systems. This project aims to build a predictive model that classifies transactions as **Legitimate** or **Fraudulent**.

The application is built using:
- Machine Learning (Logistic Regression)
- Streamlit for UI
- Data Visualization for insights

---

## ⚙️ Features

✔️ Real-time fraud prediction  
✔️ Bulk prediction via CSV upload  
✔️ Interactive UI with Dark/Light mode  
✔️ Model performance metrics (Accuracy, ROC Curve, Confusion Matrix)  
✔️ Download predictions as CSV  

---

## 🧠 Machine Learning Workflow

1. Data Preprocessing  
2. Handling Imbalanced Data (Undersampling)  
3. Train-Test Split  
4. Model Training (Logistic Regression)  
5. Model Evaluation  

---

## 📊 Model Performance

- Training Accuracy: 95.93%
- Testing Accuracy: 93.40%

## ▶️ How to Run Locally

1. Clone the repository: git clone https://github.com/Rutveefantastico/Credti_card_fraud_detection_system.git 
2. Navigate to the folder: cd credit-card-fraud-detection
3. Install dependencies: pip install -r requirements.txt
4. Run the app: streamlit run app.py
